# UI 风格规则

整体要求：简约、高级、大气。参考 Apple 设计语言。

## 布局

- 留白充足，间距统一（8px 的倍数）
- 内容居中，最大宽度受限（不铺满宽屏）
- 列表/卡片之间间距一致
- 不使用密集排列

## 颜色

- 主色调：克制的中性色（浅灰/白底），品牌色只做点缀
- 背景：分层清晰——`--bg-app`（最底层）、`--bg-content`（卡片/面板）、`--bg-hover`（悬停）
- 文字层级：`--text-primary`（标题）、`--text-secondary`（正文）、`--text-tertiary`（辅助信息）
- 不使用高饱和色，不使用彩虹色盘

## 圆角

- 容器/卡片：12px
- 按钮/输入框：8px
- 头像/徽章：50%（圆形）
- 弹窗/模态框：16px

## 字体

- 系统字体栈（`-apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`）
- 标题：600 weight
- 正文：400 weight
- 辅助文字：12-13px，`--text-tertiary` 色

## 阴影

- 卡片：轻微阴影（`0 2px 8px rgba(0,0,0,0.06)`）
- 弹窗：较大阴影（`0 8px 30px rgba(0,0,0,0.12)`）
- 悬停态：阴影略微加深
- 不使用浓厚阴影

## 动效

- transition：`0.2s ease` 统一
- 不使用夸张动画
- 加载态用简洁的旋转/呼吸动画

## 组件风格

- **按钮**：圆角 8px，填充/描边两种样式，高度统一 36px
- **输入框**：浅灰边框，聚焦时主色边框
- **列表项**：hover 有浅灰背景，无多余装饰线
- **卡片**：白色背景 + 圆角 12px + 轻微阴影
- **模态框**：半透明遮罩 + 居中卡片，关闭按钮右上角
- **分割线**：`1px solid var(--border-color)`，不加粗
- **滚动条**：细滚动条，仅悬停时显示

## 反面案例（不要做）

- 不要用亮色边框或描边强调不需要强调的元素
- 不要用 emoji 做 UI 图标（交互按钮用文字或 SVG）
- 不要使用斑马纹列表
- 不要使用全大写文字
- 不使用闪烁/旋转/弹跳动画
- 不在生产界面显示调试信息
